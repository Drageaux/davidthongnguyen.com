Tech Stack

Developing web applications is my cup of (bubble) tea.

Frontend
  • Angular: This framework has the coherently combinatory sweetness of JavaScript freedom and your creamy object-oriented coding paradigm.
  • RxJS: Most Angular consumers underestimate the importance of this Observables library, but I understand it deeply and can concoct streams of sugary asynchronous data.
  • Sass: Imagine if CSS can hold variables. That's Sass.
  • Bootstrap: The pre-built UI spice that many utilize.
  • Semantic UI: My personal favorite alternative to Bootstrap.
Backend
  • Node & Express: The behind-the-scene JS tea that everybody knows.
  • Firebase: In case we want to indulge on the creamy frontend, Google Firebase can take care of the backend. At a cost for scaling, of course. But at least we will not have to worry about reinventing realtime data services.
  • GraphQL: Whereas RESTful API delivers an entire package of flavor all at once, GraphQL refines whichever parts you as a consumer wants to taste. There are existing GraphQL code generators that pack a punch.
